## Hyperparameter optimisation: Grid scan (54:45)

Maybe I need to simplify it. I might have two different hyperparameters, such as weight regularization and bias regularization. I could test certain numbers in a fixed grid. For instance, if you have ten hyperparameters instead of two, the distances between points become much larger. To handle this, you need to add more points and conduct more experiments to create a usable grid. Additionally, some hyperparameters might span an infinite range, complicating the creation of a grid.
